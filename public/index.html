Url Shortener

Pass a url as a parameter after /shorten/ to receive a shortened URL

Examples:

https://short-url-final-beaubruderer.c9users.io/shorten/http://www.google.com

https://short-url-final-beaubruderer.c9users.io/shorten/https://www.stackoverflow.com

Example Output:

{ "original_url": "http://www.google.com", "short_url": "https://short-url-final-beaubruderer.c9users.io/x6" }

"short_url" will redirect to "original_url"
